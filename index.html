le CSS

quelle est le role du CSS ?

C est une langue informatique qui permet de mettre en forme des pages web (HTML ou XML)

Quelle est l'enjeu majeur du CSS ?

 Permettre la mise en forme hors des documents, il est par exemple possible de ne décrire que la structure d’un document en HTML, et de décrire toute la présentation dans une feuille de style CSS séparée
